Widget · Inline badge mode

Trust badges, inline with your token list

Tag any element — a span, table cell, or list item — with data-trustgate="token-shield" and data-trustgate-address="0x…". The widget scores each address in the background and drops a compact badge right next to the token name. No badge is shown for non-tradeable tokens (NTT) or when scoring fails.

This page loads the real /widget.js from the same origin. Badges appear once each address is scored by /api/widget/score/{address}.

Token list (span)

Each token name is a <span> carrying the address. The badge renders immediately after the name. No data-trustgate-style is set, so these use the default minimal style (tier only).

U USD Coin 0xA0b8…eB48
W Wrapped Ether 0xC02a…6Cc2
D Dai Stablecoin 0x6B17…1d0F

Table (td)

The ticker cell is a <td>. The badge sits inline after the ticker text within the same cell wrapper.

Plain list (li)

Works on <li> too — any element type is supported.

  • Uniswap (UNI)
  • Aave (AAVE)

Full style (score + tier)

Add data-trustgate-style="full" to show the numeric score alongside the tier, e.g. 87 · HIGH_ELITE. Omitting the attribute — or setting data-trustgate-style="minimal" — shows the tier only.

U USD Coin 0xA0b8…eB48
W Wrapped Ether 0xC02a…6Cc2
L Chainlink 0x5149…86CA

Dynamically rendered tokens

Token lists are often rendered after load. The widget watches the DOM with a MutationObserver, so rows added later are scored and badged automatically.
